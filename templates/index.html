About This AI Project

The AI-Powered Spam Email Detection System is an Artificial Intelligence application that classifies email and SMS messages into Spam or Safe categories.

The application applies Natural Language Processing (NLP) techniques to convert text into numerical features using TF-IDF Vectorization.

The processed text is then classified using the Multinomial Naive Bayes machine learning algorithm trained on the SMS Spam Collection dataset.
